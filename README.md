# A股板块主线判断与选股监控系统 V5.0

纯前端单页应用（Single-File HTML App），无需构建、无需后端，打开 `index.html` 即可使用。

## 功能

- **总览**：市场情绪温度、指数实时指标（RSI6 / MA5 / MA10）、主线 TOP5、五分类速览、今日结论
- **主线判断**：板块五阶段（启动/确认/加速/赶顶/退潮）判定、六项评分明细、8 赶顶信号 + 5 退潮信号勾选表、主线质量五分类、隔夜对比
- **选股推荐**：高盈亏比模型（止损/目标位/盈亏比/建议仓位），回避清单
- **持仓监控**：本地持仓盈亏、止盈止损纪律指导（数据存于浏览器 localStorage，不上传）
- **预警中心**：板块退潮/赶顶/超买/宽度预警，持仓止损止盈预警，历史持久化
- **指标详情 / 方法说明 / 数据源**：全部公式阈值公开，数据来源透明

## 数据

- 行情与日K线：腾讯行情接口（qt.gtimg.cn / web.ifzq.gtimg.cn，JSONP 无跨域限制）
- 资金流：东方财富 push2 接口（不可达时诚实标注"数据不足"，绝不编造）
- 内置快照由 bake_v5.py 基于真实行情烘焙
